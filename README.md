## Description

Backend service for a chat application built with NestJS, PrismaORM, MongoDB, and RabbitMQ.

This project includes user authentication with JWT, complete user profile management (including photo upload, zodiac/horoscope calculation), 
and a chat system that uses RabbitMQ for event-driven messaging and Socket.IO for real-time message delivery to clients.

## Project setup

```bash
# Install depedency
$ npm install

# Copy file environment
$ cp .env.example .env

# Run container Docker (MongoDB & RabbitMQ)
$ docker-compose up -d

# Generate prisma client
$ npx prisma generate

# Run the application NestJS
# Development
$ npm run start

# Production
$ npm run build
$ npm run start:prod

# Testing: Unit Test
$ npm run test
```
